Always on Top


Slash Command Description
help Sends this list of commands.
data Gets data related to Always on Top (i.e. Discord.js version, RAM usage).
support Sends the link to the Support Discord server.
setup Sets your server up in our Database.
configure add Adds a new pinned message to a channel.
configure remove Removes a pinned message from a channel.
configure edit Edits an existing pinned message.
configure list Sends a list of all pinned messages.
